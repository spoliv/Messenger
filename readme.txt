
Для запуска мессенджера:

1. Запускаем сервер - server.py

2. Запускаем срипт-клиент - client.py

На данный момент на сервере зарегистрировано три клиента:

test1, test2, test3 c одинаковым паролем - 123.

Для запуска клиентов запускаем скрипт client.py соответственное количество раз.

Для начала работы и активации ввода сообщений нажимаем кнопку "Чат" или даблклик 
по любому контакту в окне "Список контактов"

Для отправления сообщения в чат служит кнопка "Чат" 
Для отправления сообщения в личной переписке - "Отправить сообщение"
Для получения истории по переписке с тем или иным пользователем вводим его имя 
в поле сообщений и жмем кнопку "Найти историю переписки" 

Важно!
Парсинг командной строки в скриптах при запуске сервера и клиента реализован
с применением Click, поэтому понадобится его установить если не установлен.
